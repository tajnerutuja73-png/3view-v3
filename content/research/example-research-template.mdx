## How to Use This File

This file lives at `content/research/example-research-template.mdx`. To publish a real report, duplicate this file, rename it, and replace all the content — including the frontmatter at the top.

## Frontmatter Reference

The section between the `---` lines at the top of this file is called **frontmatter**. It controls everything that appears in listings, cards, and the report sidebar:

| Field | What it controls |
|---|---|
| `title` | The report headline |
| `slug` | The URL: `/research/your-slug` |
| `ticker` | Stock ticker shown in the header |
| `company` | Company name |
| `exchange` | Exchange (e.g. NSE, BSE) |
| `sector` | Used for coverage filtering |
| `reportType` | Initiation, Deep Dive, Update, or Sector |
| `publicationDate` | Shown in header and listings |
| `priceAtPublication` | Price on the day of publication |
| `thesis` | One-line thesis shown in the sidebar |
| `excerpt` | Summary shown in archive listings |
| `featured` | Set to `true` to surface on homepage |
| `tags` | Topic tags shown below the title |
| `pdf` | Filename in `/public/downloads/pdf/` |
| `model` | Filename in `/public/downloads/models/` |
| `readTime` | Estimated minutes to read |

## Writing the Report Body

Everything below the second `---` is the report body. You can use:

### Standard Markdown

Write paragraphs as plain text. Add a blank line between paragraphs.

Use **bold** for emphasis. Use *italics* for secondary emphasis.

### Bullet Lists

- First point
- Second point
- Third point

### Numbered Lists

1. Step one
2. Step two
3. Step three

### Blockquotes

Use blockquotes for key conclusions or highlighted observations:

> The core thesis in one memorable sentence. This renders in a larger serif font with a gold left border.

### Tables

| Metric | FY23 | FY24 | FY25E |
|---|---|---|---|
| Revenue (₹Cr) | 1,200 | 1,580 | 2,050 |
| EBITDA Margin | 18% | 21% | 24% |
| PAT (₹Cr) | 145 | 210 | 310 |

### Adding PDFs and Models

To add a PDF download button to this report:

1. Place the PDF file in `public/downloads/pdf/`
2. Set `pdf: "your-filename.pdf"` in the frontmatter above

To add a financial model:

1. Place the Excel file in `public/downloads/models/`
2. Set `model: "your-filename.xlsx"` in the frontmatter above

The download buttons will appear automatically below the report tags.
